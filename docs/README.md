# ENEE322/323 Project 6 — Butterworth RLC Pulse Response

This folder is organized so the scripts, generated figures, exported data, and Simscape notes stay separate.

## Folder layout

```text
ENEE323_Project6_Butterworth/
  code/       MATLAB scripts/functions
  figures/    saved PNG and FIG plots
  data/       MAT files generated by scripts and Simscape export
  simulink/   Simscape build notes and plotting helper
  docs/       written equations and notes
```

## How to run

In MATLAB, open the `code` folder and run:

```matlab
run_all_project6
```

This automatically does Parts (i), (ii), and (iii), writes Part (iv) Simscape notes, and runs Part (v) only after Simscape data exists.

## Part (iv) Simscape export

The comparison script expects this file:

```text
data/part_iv_simscape_output.mat
```

containing:

```matlab
t_sim
x_sim
y_sim
```

where `y_sim` is the voltage across the 100 ohm output resistor.

After exporting those variables, rerun:

```matlab
part_v_compare_if_available
```

or rerun the master script.

## Important correctness note

The Fourier coefficient

```text
X_k = sin(k alpha pi)/(k pi)
```

can be negative. The cosine reconstruction keeps the sign of `X_k`; it does not replace `X_k` with `abs(X_k)` unless the missing phase shift is added.
